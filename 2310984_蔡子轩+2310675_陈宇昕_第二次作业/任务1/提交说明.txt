推荐系统评分预测作业提交说明

一、目录结构
1. 可执行文件：包含生成最终预测结果和复现实验的脚本入口。
2. 实验报告：包含 PDF 实验报告，以及 tex 源文件和图片资源。
3. 实验结果：包含最终预测文件 final_result.txt 和主要实验 CSV。
4. 源码：包含完整 Python 源码、数据文件、依赖说明和关键结果文件。

二、最终结果文件
最终预测结果为：
实验结果/final_result.txt

该文件已按 Test.txt 的用户块顺序和物品顺序生成，共 610 个用户块、9982 条预测评分，评分范围限制在训练集评分范围内。

三、运行方法
进入“可执行文件”目录后：
Linux / WSL / macOS 使用：
bash generate_result.sh

Windows 使用：
generate_result.bat

四、模型说明
最终提交结果使用三模型加权融合：
0.43 * ItemCF + 0.45 * 48维 Matrix Factorization + 0.12 * UserCF

Test.txt 不包含真实评分，因此报告中的 RMSE 来自 Train.txt 内部交叉验证。
